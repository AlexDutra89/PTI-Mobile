# PTI - App de Delivery (React Native)

Este projeto faz parte da minha PTI na disciplina de Programação para Dispositivos Móveis no Senac EAD. 

### O Desafio
O objetivo foi criar um componente funcional reaproveitável para um app de delivery. Foquei em construir uma estrutura flexível que recebe todas as informações (nome, preço, calorias e imagem) via JSON, seguindo as boas práticas de componentização do React Native.

### Tecnologias e Conceitos
* React Native e Expo para a estrutura do app.
* Props para tornar o componente dinâmico.
* JavaScript para a lógica e manipulação dos dados.

### Onde está o código?
* O componente solicitado pelo enunciado está em ProductCard.js.
* Em App.js, realizei a simulação do cardápio enviando os dados para o componente.
